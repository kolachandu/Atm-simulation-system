# ATM Simulation System

A simple Java console application that simulates common ATM operations.

## Features

- Card number and PIN login
- Balance inquiry
- Cash deposit
- Cash withdrawal
- Fund transfer between accounts
- PIN change
- Mini statement / transaction history

## Run

Compile:

```bash
javac -d out src/*.java
```

Run:

```bash
java -cp out Main
```
